## Вопросы и ответы

### Почему нет шелл-скриптов и мейк-файлов?

Всё написано на Go, включая скрипты. Введение другого языка добавляет платформенную зависимость и сложность. Скрипты на Go портативны и надёжны.

Сборка выполняется командой:

```bash
cd scripts && go run . build
```

Кэш сборки Go делает инкрементальные сборки быстрыми. Мейкфайлы не нужны.

### Можно ли использовать библиотеку X?

Нет. Проект избегает сторонних зависимостей. Используйте только стандартную библиотеку Go.
